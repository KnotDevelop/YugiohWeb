##### ภาพรวม #####

เป็นเว็บแอปพลิเคชันที่ให้ผู้ใช้สามารถเรียกดูและค้นหาการ์ด Yu-Gi-Oh! ได้โดยใช้ตัวกรองและการจัดเรียงที่หลากหลาย 
ผู้ใช้สามารถสำรวจการ์ดตามประเภท, เผ่าพันธุ์, ประเภทกรอบ และอื่นๆ อีกมากมาย เว็บไซต์ยังมีฟีเจอร์สำหรับดู Archetype, 
ชุดการ์ด และการสุ่มการ์ด

##### ฟีเจอร์ #####

1. การค้นหาและการกรองการ์ด

ดูการ์ด Yu-Gi-Oh! ทั้งหมดในฐานข้อมูล

ใช้ตัวกรองเพื่อจำกัดการค้นหาโดย:

ประเภท (Monster, Spell, Trap, ฯลฯ)

เผ่าพันธุ์ (Dragon, Warrior, Fiend, ฯลฯ)

ประเภทกรอบ

การจัดเรียง: ATK, DEF, ชื่อ, ประเภท, ระดับ, ID

ค้นหาการ์ดโดยใช้แถบค้นหา

คลิกที่การ์ดเพื่อดูรายละเอียดเพิ่มเติม

2. หน้า Archetype

ดูการ์ดทั้งหมดที่เกี่ยวข้องกับ Archetype ใดๆ

3. หน้า Card Set

เรียกดูชุดการ์ด Yu-Gi-Oh! ที่เคยวางจำหน่ายทั้งหมด

4. หน้า Random Card

สุ่มแสดงรายละเอียดของการ์ด Yu-Gi-Oh! 1 ใบแบบสุ่ม

##### การติดตั้ง #####

หากต้องการรันโปรเจคนี้บนเครื่องของคุณ:

โคลน repository:

git clone https://github.com/KnotDevelop/YugiohWeb.git

เข้าไปยังโฟลเดอร์โปรเจค:

cd YugiohWeb

ติดตั้ง dependencies:

npm install

เริ่มเซิร์ฟเวอร์สำหรับพัฒนา:

npm run dev

##### เทคโนโลยีที่ใช้ #####

Frontend: React Vite

API: ใช้ Yu-Gi-Oh! Card API: https://ygoprodeck.com/api-guide/
